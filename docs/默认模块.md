## GET [Kimi] 余额

GET /v1/users/me/balance

### 请求参数

| 名称          | 位置   | 类型   | 必选 | 说明 |
| ------------- | ------ | ------ | ---- | ---- |
| Authorization | header | string | 是   | none |

> 返回示例

> 200 Response

```json
{
    "code": 0,
    "data": {
        "available_balance": 48.7099,
        "voucher_balance": 0,
        "cash_balance": 48.7099
    },
    "scode": "0x0",
    "status": true
}
```

### 返回结果

| 状态码 | 状态码含义                                              | 说明 | 数据模型 |
| ------ | ------------------------------------------------------- | ---- | -------- |
| 200    | [OK](https://tools.ietf.org/html/rfc7231#section-6.3.1) | none | Inline   |

### 返回数据结构

状态码 **200**

| 名称                 | 类型    | 必选 | 约束 | 中文名 | 说明                                                         |
| -------------------- | ------- | ---- | ---- | ------ | ------------------------------------------------------------ |
| » code               | integer | true | none |        | 响应码。0 表示成功。                                         |
| » data               | object  | true | none |        | none                                                         |
| »» available_balance | number  | true | none |        | 可用余额（单位：人民币元），包含现金余额和代金券余额。当小于等于 0 时，用户无法调用推理 API |
| »» voucher_balance   | integer | true | none |        | 代金券余额（单位：人民币元），不可为负                       |
| »» cash_balance      | number  | true | none |        | 现金余额（单位：人民币元），可为负值表示欠费。当为负值时，available_balance 等于 voucher_balance 的值 |
| » scode              | string  | true | none |        | none                                                         |
| » status             | boolean | true | none |        | none                                                         |
